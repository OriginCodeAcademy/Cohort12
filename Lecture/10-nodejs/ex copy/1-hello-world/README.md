# Ex1: Hello World

## Tasks

- [ ] Write the following JavaScript inside `app.js`

```js
function add(a, b) {
    return a + b;
}

console.log(add(1,1));
console.log(add(1,2));
console.log(add(1,3));
console.log(add(1,4));
console.log(add(1,5));
```

- [ ] In the command line, `cd` into this folder and type `node app.js` to run the code.
